JSP自定义标签

JavaEE    Coselding    2014-12-26 11:44

本文为博主原创，允许转载，但请声明原文地址：http://www.coselding.cn/article/2014-12-26/131078057.html

1、实现Tag接口或者继承TagSupport，实现一个标签处理器类，将要在JSP中出现的java代码放到这个处理器类中；
  doStartTag：标签开始处调用
  doAfterBody：标签内容体处调用
  doEndTag：标签结束处调用
2、编写tld文件，WEB-INF目录下（服务器启动会把WEB-INF下的所有tld加载到内存中），把标签处理器类描述成一个标签，并声明一个名称空间（和Schema一样）
 A、绑定一个uri
 B、定义标签处理器类
    <tag>
      <name>showTest</name>
      <tag-class>com.silence.jsp.I18nTag</tag-class>
      <body-content>empty</body-content>
    </tag>
3、在JSP页面中声明<%@ taglib>导入tld声明的命名空间即可使用；
4、功能：
  A、控制JSP页面中的任意部分的内容的输出（一部分或整个页面），由doStartTag的返回值控制，doEndTag控制标签后的代码是否执行
  B、控制JSP页面中的内容重复输出，IterationTag的doAfterBody方法返回值控制重复执行，重复执行的是标签体，开始结束标签不能重复执行；
  C、提取标签体数据，进行修改后输出：doStartTag返回获取标签体的标记，BodyTag的getBodyContent获取标签体内容；
5、SimpleTag简单标签的使用：
  SimpleTag接口或者SimpleTagSupport类继承
  getJspBody：获取标签对象JspFragment，
           其invoke方法将标签实体写入到一个Writer中，Writer为null表示写入
            pageContext的Out，可以通过此处将标签体写入到一个字符缓冲中，达到标签
            内容的提取功能
  doTag：表示标签执行方法，所有的标签处理代码在此方法中进行
  A、提取标签体：
    // 得到标签体对象
    JspFragment jf = this.getJspBody();
    // 将标签体写入到缓冲中
    StringWriter stringWriter = new StringWriter();
    jf.invoke(stringWriter);
  B、重复执行：
  for (int i = 0; i < 5; i++) {
     getJspBody().invoke(null);
  }
  C、控制输出
  // 抛出忽略标签后JSP代码的异常，实现对JSP页面的控制输出
  throw new SkipPageException();
6、标签属性编写：
  A、标签处理器类中编写属性名的setter方法
  B、tld中声明属性：
  <attribute>
     <name>count</name>//表示属性名称
     <required>true</required>//表示该属性属否必须
     <rtexprvalue>true</rtexprvalue>//表示是否支持运行时表达式
                                   //（EL表达式，脚本变量）
  </attribute>
  C、属性之间通过反射进行赋值，支持八种基本数据类型，其它类型需要注册类型转换器
7、打包自己的标签库
把标签处理器类放在一个JavaProject的包下
新建META_INF文件夹，把tld文件放到该文件夹下
Emport--jar file--只留包和META-INF--导出
